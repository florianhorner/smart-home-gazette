I The Status Quo

The ecosystem that wasn't actually a problem

It worked. Really. IKEA Tradfri for lights and smart plugs. Tado for thermostats—its own ecosystem, its own app, does the job. Assorted plugs from other manufacturers. Alexa as a shared roof over everything, incomplete but present.

“I had four apps. But each one did what it was supposed to,” the subject explains in retrospect. “I wasn't unhappy.” This is important to understand: the starting position was stable. No emergency, no crisis. Just a fragmented setup that most people would have accepted as the normal state of things.

The crucial detail: every one of these choices was individually reasonable. IKEA lights are good. Tado is excellent for heating. The plugs work. Nobody buys products in anticipation of future ecosystem conflicts.

II The Catalyst

One sensor. One thought. One search query.

The Aqara FP2—millimeter-wave presence detector, HomeKit-compatible, promises multi-zone detection. The pitch: the light doesn't just know whether someone is in the room, but where. “Not whether. Where.”

Apple Home seemed like the logical platform. The FP2 is native HomeKit. Clean. One app. End of story—in theory.

⚠️
The problem: IKEA Tradfri, Tado, and the assorted plugs responded to Apple Home with the enthusiasm of a cat toward water. Not all devices supported. Not fully. Not without buying everything again.

“Either I buy everything new and migrate the whole thing—or I don't,” he says. He chose “don't.” The return to Alexa wasn't a defeat but a conscious, pragmatic decision: keep the infrastructure that works.

“Alexa wasn't the problem. Alexa was the sensible answer. The problem was that the FP2 was sitting on the table and I knew it could be better.”
— Subject, 3:47 AM

Then: a Google search. “Home Assistant IKEA Tado integration.” Twenty-seven tabs. A Reddit thread. A YouTube playlist. A Discord server with 40,000 members.

🔴
Editorial warning: This search statistically leads to Home Assistant. We recommend performing it only with adequate free time available. Total duration to first configuration.yaml: 72 hours.
III The Descent

Home Assistant: The platform that can do everything—and proves it

Home Assistant delivers on its promise. IKEA Tradfri: integrated. Tado: integrated. Plugs: integrated. Aqara FP2: integrated. Everything in one interface, one app, one dashboard. The original goal: achieved.

What nobody communicates in advance: “everything integrated” means “you now understand how everything works.” That is not the same thing.

The decisive step: a device in an upstream project was misbehaving. Chinese preset names crashed the MQTT bridge—a unicode handling bug in Rust. Someone had filed the issue. Nobody had fixed it. The subject read the thread. Then the code. Then he opened Claude Code.

“I had never written Rust before. But Claude Code read the codebase and the fix became obvious within an hour. So I submitted it.” The PR was merged. The repository has several thousand stars. His name is in the changelog now.

🌉
The escalation continued. After the Rust fix, he found a stalled Home Assistant add-on—someone had asked “Is this project abandoned?” He upgraded its core dependency, wired up MQTT export, hardened its security headers, and published a fork. In Bash. A language he also doesn't write. Then he submitted PRs to the original repo. For a light switch.
✔️
Current status: The lights work. Multi-zone presence detection runs. Tado integrated. All plugs active. The FP2 knows where in the room someone is. This was always the goal. The path there was optional.

When asked if he'd do it again, he looked at the screen where a compiler error glowed. Then at the light, burning quietly. “Ask the maintainer.”

IV The Aftermath

The man who just wanted to turn on a light now has a GitHub profile

The subject—an account executive by profession, not an engineer—now has pull requests across three programming languages he doesn't professionally write: Rust (unicode crash fix, merged), Bash/Docker/Nginx (add-on modernization + MQTT export), and TypeScript (Lovelace card). All in projects he actually depends on. All done with Claude Code.

The pattern, he insists, is not “I learned to code.” The pattern is: “The barrier to contributing to open source is gone.”

“I don't write Rust. I don't write Bash. I don't write TypeScript. But I use software that's written in all three, and when it broke, I fixed it. That's the whole story.”
— Subject, now fully awake at 4:12 AM

His parents still don't understand. The maintainers do.
